一、核心结论

⚡ 三句话总结

1. 2026年3月5日，OpenAI 发布 GPT-5.4，成为首个带原生计算机操作（Computer Use）能力的通用大模型，OSWorld 75.0%（人类水平 72.4%），1M 上下文窗口，定位"专业工作"——标志着 AI 从"生成内容"正式进入"操作计算机执行任务"阶段。
2. 国产模型阵营同期集中爆发——GLM-5 开源登顶编程榜、字节 Seed 2.0 系列全线覆盖文本/代码/图像/视频、MiniMax M2.5 SWE-bench 80.2%，在编程和 Agent 场景已具备替代国际模型的能力。
3. 大模型正从"单模态文本生成"加速向"多模态全栈能力"演进，各家路线分化明显——OpenAI 专业工作流+Computer Use、Anthropic 深度编程+Agent Teams、Google 全栈多模态、字节 内容生成全链路。个人用户的最优策略是"主力+备选组合，按场景切换"（详见第三章 模型对比及选型建议）。

• 闭源旗舰的成本差异巨大：Opus 4.6（$5/$25）是闭源中最贵的，GPT-5.4（$2.50/$15）性价比更优，但 Opus 编程更强

3.3 各模型深度解读

每款模型都有独特的"灵魂"和明确的短板。以下将 Benchmark 数据与用户真实评价融合——数据说"能力有多强"，用户说"体验有多好"。

① Claude Opus 4.6

编程能力数一数二

LFG Bench 9.25/10（实战编程第一），ARC-AGI-2 68.8%（抽象推理遥遥领先，较 4.5 的 37.6% 提升 83%）。MRCR v2 76%，长上下文代码检索 4.1x 提升，1M 上下文(Beta)可以一次性理解整个代码库。

🗣️ 用户声音："Opus 4.6 is the new king for hard problems. I threw a Next.js e-commerce project at it and it completed every feature including checkout — Codex 5.3 missed the checkout flow entirely."
—— LFG Bench 实测对比，HN 讨论帖（1561 赞）

行业的引领者和规则制定者

Anthropic 不只是做模型，而是在定义 AI 的未来形态——MCP（模型上下文协议）已被 Cursor/Windsurf/Zed 等主流 IDE 原生支持，正在成为 AI Agent 的事实标准接口。Skills、Todo、Agent Teams 都是他们率先提出并实现的。

🗣️ 用户声音："Agent Teams 是杀手级功能。16 个 Agent 从零构建 C 编译器——AI 编程不再是'帮我写函数'，而是'帮我做项目'。"
—— X/Twitter 开发者评论

最有品味、最具灵魂的 AI

AI 宪法（Constitutional AI）让 Claude 在安全与能力之间找到了优雅的平衡。它不只是一个工具，而是一个有自己思考方式的伙伴。公司员工留存率在 AI 行业中名列前茅，说明这家公司有真正的使命感和文化。

价格昂贵且封号严格

定价 $5/$25 是闭源中最贵的（GPT-5.4 仅 $2.50/$15，贵一倍）。封号政策严格，违规即封无申诉。技术文档写作质量相比 4.5 有退化——"编码用 4.6，写作留 4.5"。月费 ≈ ¥216（按 200万 token 计），是 DeepSeek 的 22 倍。

② GPT-5.4

Computer Use 首个超越人类的通用 AI

OSWorld 75.0%（人类水平 72.4%），首个带原生计算机操作能力的通用大模型。能直接操作 Excel 做财务建模、浏览网页、控制桌面应用。BrowseComp 89.3%（Pro），投行基准 87.5%，深度 Excel/Sheets 集成。

🗣️ 用户声音："Computer Use 是真正的游戏规则改变者。它能直接在我的电脑上操作 Excel 做财务建模。OSWorld 75% 超人类不是营销话术——这是实打实的。"
—— 早期 ChatGPT Pro 用户

最全能的 AI，什么都能干什么都不差

GPQA 93.2%+ 科学推理顶级，1M 上下文窗口，文/图/音/视/CU 五模态全覆盖。推理、编程、多模态全面无短板，是真正的"全能选手"。工具搜索减少 47% token 消耗。

不封号 + 适合长任务，最省心的选择

你可以放心把一个 8 小时的工作流交给它，不会半路被掐断。幻觉降低 33% 是最被低估的改进——做事实核查和报告生成时，这比任何 benchmark 提升都重要。

🗣️ 用户声音："幻觉降低 33% 是最被低估的改进。做事实核查和报告生成的场景下，这比任何 benchmark 提升都重要。"

编程深度不如 Opus，Pro 版定价极高

开放性编程不如 Opus（LFG Bench 7.5 vs 9.25）；Pro 版 $30/$180 定价极高，普通人用不起；>272K 上下文翻倍计费。标准版 $2.50/$15 性价比尚可。

③ Gemini 3 Pro

多模态感知力最强，原生视频处理独家

原生文/图/音/视四模态不是后期缝合，而是底层架构特性。原生视频流处理是独家能力，1M+ 上下文可以一次性分析整部电影或数小时的会议录像。Arena 排名第 3（~1435），GPQA ~87%。

Google 生态深度集成，护城河难以复制

与 Google 搜索/Maps/Workspace 的深度集成构建了其他厂商难以复制的护城河。做前端设计和视觉任务时灵感四溢，适合内容创作者和多媒体工作者。

Flash-Lite 定义轻量模型新标准

363 tok/s + $0.25/M，速度是 GPT-5.3 Instant 的 5x，价格是其 1/2。所有轻量任务的绝对最优选择。

🗣️ 用户声音："Flash-Lite 性价比逆天，363 tok/s 的速度让我把所有轻量任务都迁过来了。但纯文本深度推理还是不如 GPT/Claude。"
—— 开发者评价

编程能力短板明显，国内访问受限

编程能力不如 Opus/GPT-5.4（尤其后端/算法题）；中文生态不如国产模型；API 在国内访问受限，需要科学上网。

④ Grok 4

Arena 排名第一的推理怪兽

Arena ~1450+（全场第一），AIME 2025 数学竞赛 93 分。凭借 10 万张 H100 的 Colossus 集群和 X/Twitter 实时数据独占优势，在推理和信息时效性上做到天花板。

X 生态绑定，信息时效性最强

实时接入 X/Twitter 全平台数据，回答的时效性远超其他模型。适合追踪实时热点、金融市场动态、突发新闻等场景。

$3000/年天价 + 擦边争议，普通人用不上

$3,000/年订阅制，性价比极低，把大多数人挡在门外。仅文/图两种模态。因"擦边"内容生成（NSFW 宽松策略）饱受争议。编程/Agent 不是强项，API 封闭。它可能是"最强但最难用"的模型。

⑤ GLM-5

开源编程最强，对齐 Claude Opus 4.5

MoE 744B（40B 激活），28.5T 训练数据。全球权威 Agent 榜单第 4、开源第 1。编程能力较 GLM-4.7 提升 >20%，在开源模型中编程能力独占鳌头。

🗣️ 用户声音："GLM-5 让我震惊——一个开源模型在编程上能接近 Opus 4.5 的水平，还能跑在国产芯片上。"
—— 腾讯网引用开发者

首个适配 7 家国产芯片的旗舰模型，信创担当

适配昇腾/海光/摩尔线程/寒武纪/昆仑芯/沐曦/燧原，首次证明国产芯片可以跑旗舰级大模型。对中国 AI 产业自主可控具有战略意义，已成为国产算力生态的核心模型供应商。

ZCode 平台让模型变工具链

配套推出 ZCode 多智能体编码平台，从单纯的"模型"变成完整的"开发工具链"。开源 + 工具链 + 国产芯片适配，走出了独特路线。

多模态和国际影响力仍有差距

多模态仅文/图/代码（无音视频）；国际影响力有限；Arena 排名中游；部分国产芯片上推理速度仍有差距。

⑥ 字节 Seed 2.0

国产首个 Arena 全球前十，真实体验驱动

Arena Text 国产第一/全球前十，Seed 2.0 Pro 准确率 76.5%。字节做大模型的逻辑和其他家完全不同——以体验为驱动而不是以刷分为驱动，强调指令遵循和长尾知识，面向真实世界的复杂工作流优化。

文/代码/图/视频全链路覆盖，内容创作利器

依托抖音/TikTok/剪映/飞书的庞大用户群和丰富场景，形成服务内容生态的完整工具链。Seedance 2.0 是首个明确商业化的 AI 视频模型（$0.14/秒），多镜头叙事+原生音频。

Pro 版推理慢且贵，海外影响力有限

Pro 推理时间 300秒+、成本翻 3 倍；视频模型商业化还在早期；海外影响力不如 OpenAI/Anthropic。

⑦ MiniMax M2.5

Agent 和工具调用能力全球前列

SWE-bench 80.2%（逼平 Claude Sonnet 4.5 的 82%），BFCL 76.8%（工具调用），BrowseComp 76.3%（Web Agent）。复杂任务速度快 37%，100 TPS 仅 $1/小时。

成立 3 年逼平国际旗舰，垂直深耕路线验证

一家成立仅 3 年、员工不到千人的公司，在 SWE-bench 上逼平 Claude Sonnet 4.5——证明了垂直深耕 + 工程效率优化的路线可以弯道超车。2025 年收入同比增长 158%，ARR 超 $1.5 亿，是中国 AI 创业公司中最会赚钱的。

通用对话不是强项，品牌知名度有限

通用对话能力不如旗舰；品牌知名度有限，很多人没听过；多模态仅文/代码/Agent，没有图像和视频能力。

⑧ DeepSeek-V3

以一己之力改写大模型成本预期

MIT 开源 + 中文代码最佳，穷人的 GPT-5

MoE 685B（37B 激活），MIT 开源让任何人都能本地部署。R1 开源后 2 周被 20+ 团队微调，用强化学习替代标注数据的范式创新影响深远。中文代码生成在社区公认为最佳。

🗣️ 用户声音："性价比之王，穷人的 GPT-5。中文代码生成最佳。"
—— 中文技术社区普遍共识

多模态极弱，高峰期排队严重

多模态极弱（仅文/代码）；R1 高峰期"系统繁忙"排队严重，体验不稳定；Arena ~1390 与旗舰有差距。

⑨ Qwen3.5-Plus

使用成本趋近于零，月费仅 ¥2.2

$0.11/$0.11 的定价意味着按 200 万 token 计月费仅 ¥2.2，是 Opus 4.6 的 1/98。对预算敏感的个人用户、学生和轻度使用者是绝对首选。

阿里云生态加持，国内分发优势

依托阿里云生态，在中国市场有天然的分发和集成优势。文/图/代码三模态，日常使用完全够用。

旗舰能力与顶尖有差距

旗舰能力与国际顶尖有差距；Agent/工具调用不是强项。适合"够用就好"的用户，不适合追求极致的场景。

四、模型演进趋势分析

趋势一：各家模型全面趋近 — 能力、模态、排名正在收敛

核心判断

2026 年大模型行业最显著的特征不是某一家遥遥领先，而是所有头部玩家正在加速趋同。这种趋近发生在三个层面：① 模态覆盖——从纯文本走向文/图/音/视/代码全模态已成标配；② 评测排名——Arena 前 5 名差距仅 1.7%，人类盲测已难以区分；③ 核心能力——推理、编程、Agent 等关键赛道各家差距急剧缩小。差异化窗口正在关闭，竞争正从"谁更强"转向"谁更快、更便宜、生态更深"。

趋势四：蒸馏小模型与端侧部署 — AI 从云端走向终端

2025–2026 年，大模型厂商纷纷将旗舰能力"蒸馏"到 1B–10B 参数的小模型中，使其能在手机、PC、IoT 等端侧设备上本地运行。这标志着 AI 部署范式从"纯云端 API"正在转向"云边端协同"——延迟更低、隐私更强、成本趋零。

论据 3：端侧部署的三重商业价值

  • 延迟归零：端侧推理不依赖网络，响应时间 <100ms，适合实时翻译、语音助手、AR/VR 交互

  • 成本归零：推理在用户设备上完成，厂商无需承担 GPU 推理成本，可实现"用户量增长但边际成本不增"的理想模型

大模型的迭代节奏正在经历从"年更"→"季更"→"月更"的加速。头部厂商的发布周期已从 2023 年的 12–18 个月缩短至 2026 年的 2–4 个月，竞争压力和技术扩散共同驱动了这一趋势。发展路径呈现"基座模型大版本 + 持续小版本优化"的双轨制。

论据 3：加速的驱动力与天花板

  • 驱动力：① 竞争压力——任何一家的突破都迫使其他家加速跟进；② 技术扩散——论文和开源使核心技术从发明到被复现的周期从 2 年缩短到 3–6 个月；③ 工程工具链成熟——训练框架（Megatron-LM、DeepSpeed）和评测体系的完善降低了迭代门槛
  • 天花板：① 高质量训练数据日趋枯竭，合成数据尚未完全替代真实数据；② 前沿模型训练成本仍在 $100M–$1B 量级（GPT-5 预估 $500M+）；③ 对齐和安全评估需要时间，不能无限压缩

趋势六：蒸馏降低门槛，但后来者追赶仍受三大核心约束

核心判断

知识蒸馏使后来者可以用 1/20 甚至 1/100 的成本获得前沿模型 80%+ 的能力，这是历史性的"后发优势"。但追赶到最后 20% 的成本是非线性增长的，核心瓶颈不在算法，而在数据飞轮、算力规模和人才密度三个维度。已经拥有强大基座模型的厂商在蒸馏上具备天然优势——它们既是"老师"又是"学校"。

论据 3：拥有基座模型的厂商在蒸馏上的结构性优势

已经拥有强大基座模型的厂商（OpenAI、Anthropic、Google、阿里、DeepSeek）在蒸馏上具备自我增强的飞轮效应：

  • Teacher 质量最高：自家旗舰模型就是最好的老师，无需依赖外部模型的输出
  • 蒸馏数据更丰富：可以控制 Teacher 生成任意领域、任意难度的训练数据，覆盖 edge case
  • 端到端优化：可以同时优化 Teacher 和 Student 的架构，如 Qwen3.5 系列从 0.8B 到 Plus 的全矩阵布局
  • 品牌信任：用户更信任"来自 GPT-5 的蒸馏"而非"来自第三方的蒸馏"

结论：蒸馏降低了入场门槛（从 $100M 降到 $5M），但并未消除头部优势。后来者可以快速达到 80 分，但从 80 分到 95 分的差距仍需要数据飞轮、算力投入和人才密度的长期积累。这就是为什么 DeepSeek 虽然用极低成本做出了 GPT-4 级模型，但在 Arena 排名上仍与 GPT-5.4 / Opus 4.6 有明显差距。

趋势七：从"选最强"到"选最懂我" — 品味、个性化与上下文成为新护城河

核心判断

当模型能力高度趋同（趋势一），用户的选择逻辑正在发生根本转变：不再是"哪个模型跑分最高"，而是"哪个模型最懂我"。差异化的核心正从模型参数转向品味（Taste）——即训练者如何筛选数据、定义对齐规则、塑造模型人格。与此同时，Memory、Rules、Projects、CLAUDE.md、Heartbeat/Cron 等个性化机制的出现，使模型能够"记住"用户并持续进化，构建起强大的迁移壁垒和用户粘性。

论据 1：品味差异化——以 Claude 的宪法 AI 为典型案例

Anthropic 的 Constitutional AI（宪法式 AI）是"品味决定差异"的最佳注脚：

  • 宪法定义行为边界：Claude 的行为由一组人工编写的"宪法原则"（如"选择最有帮助且最无害的回答"、"如果不确定，宁可谨慎也不要臆造"）而非纯粹的 RLHF 驱动。这使得 Claude 在主观评价中被广泛认为"更有品味"——回答更谨慎、更有深度、更像一个有原则的合作伙伴
  • 对比 GPT 的 RLHF 路线：OpenAI 更依赖大规模人类反馈（每日 1 亿+ 用户交互的隐式反馈），这使 GPT 在"取悦多数人"方面表现更好，但有时被批评为"过于讨好"（sycophantic）。两种路线没有绝对优劣，但产生了明显的人格差异
  • Anthropic 公开系统提示词：2024 年 Anthropic 成为全球第一家公开 AI 模型完整系统提示词的公司，展示了 Claude 的"灵魂"如何被定义。这种透明度本身就是差异化策略

"品味"的本质：当两个模型在 MMLU 上都达到 90%+ 时，用户感知到的差异并非来自参数或架构，而是来自训练数据的筛选标准、对齐原则的设计哲学、以及模型在模糊地带的取舍方式。就像同样原料做出的菜，厨师的品味决定了最终的味道。

Auto Memory模型自动从对话中提取并记录用户偏好Claude Code Auto Memory（自动写入 CLAUDE.md）零摩擦积累个性化数据，用户无感知中越来越"被理解"

论据 4：从工具到伙伴——"最懂我"的商业价值

这些个性化机制的叠加效应正在改变 AI 产品的竞争逻辑：

  • 迁移成本指数增长：Memory Import 虽然降低了"记忆搬家"的门槛，但用户在 Projects 中积累的工作流、在 Rules 中定义的行为模式、在长期交互中形成的隐性偏好，远比结构化记忆更难迁移
  • 使用时长 → 数据壁垒：ChatGPT 每日 1 亿+ 活跃用户意味着每天产生海量的交互数据，这些数据反过来优化模型的个性化能力。Claude 通过 Memory Import 抢夺存量用户，但增量数据仍是 OpenAI 的优势
  • 选择标准的范式转移：企业采购正在从"选跑分最高的"转向"选最适合我们工作流的"——这包括模型是否支持自定义 Rules、是否有 Project 级上下文管理、是否能通过 MCP 接入内部工具链

终极趋势：模型竞争正在从"智力竞赛"（谁更聪明）转向"关系竞赛"（谁更懂我）。如同手机从"比参数"到"比生态"的演进，AI 模型的最终护城河不是参数规模，而是对每个用户的理解深度。选择排名第一的是一种策略，选择性价比最高的是另一种策略，但越来越多的高价值用户正在选择第三条路：选择最懂自己的。
